# 动态人格模块构想

> **构思时间**: 约 2023 年
> **整理时间**: 2026-02-07
> **来源**: 手写笔记本

---

## 💡 核心灵感

人格不是一成不变的，而是**固定基础 + 动态成长**的组合。

---

## 🔑 关键设计点

### 1. 固定 vs 动态

| 类型 | 说明 | 示例 |
|------|------|------|
| **固定部分** | 基础性格，不太变化 | 内向/外向、乐观/悲观 |
| **动态部分** | 会被经历影响 | 信任度、亲密度、警惕性 |

### 2. 九大人格因子

可能的因子设计（参考心理学模型）：

| 因子 | 说明 | 动态/固定 |
|------|------|----------|
| 开放性 | 对新事物的接受度 | 半动态 |
| 尽责性 | 做事认真程度 | 固定 |
| 外向性 | 社交倾向 | 固定 |
| 宜人性 | 友善程度 | 动态 |
| 神经质 | 情绪稳定性 | 动态 |
| 信任度 | 对用户的信任 | 动态 |
| 亲密度 | 关系亲近程度 | 动态 |
| 自信心 | 自我评价 | 动态 |
| 好奇心 | 探索欲望 | 半动态 |

### 3. 记忆-人格联动 ⭐

> **"让记忆因子影响人格"**

这是最核心的设计：

```
┌─────────────┐     影响      ┌─────────────┐
│   记忆模块   │ ──────────→  │   人格模块   │
└─────────────┘              └─────────────┘
     │                              │
     │  正面记忆 → 提升亲密度/信任度   │
     │  负面记忆 → 降低信任度/提高警惕 │
     │  长期相处 → 人格逐渐成长       │
     └──────────────────────────────┘
```

### 4. 动态变化维度

笔记中提到的动态变化：
- **成长**: 随时间和经历成长
- **提醒**: 可被事件触发
- **快速**: 即时响应某些刺激
- **优势**: 在某些领域形成专长

## 🆚 与现有 PsycheEngine 对比

| 维度 | 笔记构想 | 现有实现 |
|------|----------|----------|
| 人格因子 | 九大因子 | 5个维度 |
| 固定+动态 | ✅ 明确区分 | 只有 baseline+value |
| 记忆影响 | ✅ 核心设计 | ❌ 无连接 |
| 长期成长 | ✅ 会成长 | ❌ 只有刺激-衰减 |
