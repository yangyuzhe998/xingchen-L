# 服务器购买与避坑指南 (Server Buying Guide)

为了让“星辰-V”稳定运行，你需要一台 **云服务器 (VPS)**。这就好比在云端租了一台永远不关机的电脑。

## 1. 推荐配置 (Configuration)

对于本项目 (Python + Docker + NapCatQQ + ChromaDB)，建议的最低配置如下：

*   **CPU**: **2核** (1核勉强能跑，但处理消息快了容易卡死)
*   **内存 (RAM)**: **2GB** (推荐 4GB。因为 ChromaDB 向量数据库和 NapCatQQ 都比较吃内存)
*   **硬盘**: **40GB SSD** (系统+Docker镜像+日志，40G足够)
*   **操作系统**: **Ubuntu 22.04 LTS** (最推荐，教程最多，兼容性最好)

## 2. 服务商推荐 (Providers)

### 方案 A: 国内大厂 (稳定、速度快，适合新手)
*   **腾讯云 (Tencent Cloud) / 阿里云 (Aliyun)**
    *   **优点**: 速度快，控制台中文友好，学生或新用户通常有“99元/年”的特惠机（轻量应用服务器）。
    *   **缺点**: 如果你想直接访问 OpenAI/DeepSeek 的国际节点，可能需要配置代理（或者使用国内的中转 API 地址）。
    *   **推荐产品**: **轻量应用服务器 (Lighthouse)**。

### 方案 B: 海外服务商 (免折腾网络，适合有经验者)
*   **RackNerd / CloudCone** (廉价 VPS)
    *   **优点**: 极其便宜 (往往 $20/年 左右)，网络直通 OpenAI，不需要配代理。
    *   **缺点**: 到了晚高峰网络可能会卡，控制台全英文。
*   **DigitalOcean / Vultr** (按小时计费)
    *   **优点**: 随时开随时删，适合测试。
    *   **价格**: 大约 $10-$12 / 月 (2GB 内存款)。

## 3. 购买步骤 (以腾讯云轻量应用服务器为例)

1.  **注册/登录**: 访问腾讯云官网，用微信扫码登录。
2.  **搜索产品**: 搜索“轻量应用服务器”。
3.  **选择套餐**:
    *   找“新用户专享”或“特惠”区域。
    *   选择 **2核 2G** 或 **2核 4G** 的配置。
    *   地域：选择离你近的（如上海、广州）。
4.  **选择镜像 (系统)**:
    *   **非常重要**：选择 **系统镜像** -> **Ubuntu 22.04 LTS**。
    *   *不要选 CentOS，也不要选预装 Docker 的应用镜像（版本可能老旧），我们自己装最稳。*
5.  **付款**: 支付宝/微信支付。
6.  **重置密码**:
    *   购买成功后，进入“控制台”。
    *   点击你的服务器卡片。
    *   找到“重置密码”，设置一个复杂的 root 密码（例如 `XingChen@2025!`）。
    *   记住公网 IP 地址 (例如 `123.45.67.89`)。

## 4. 连接服务器

在你的电脑终端 (PowerShell 或 CMD) 输入：
```bash
ssh root@你的公网IP
# 例如: ssh root@123.45.67.89
```
输入刚才设置的密码，看到 `root@VM-xxx:~#` 提示符，就说明你已经成功登录这台云端电脑了！

接下来就可以按照 `deployment.md` 中的步骤开始安装 Docker 了。
